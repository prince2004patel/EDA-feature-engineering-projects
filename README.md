# 📊 EDA & Feature Engineering Projects

A collection of **Exploratory Data Analysis (EDA)** and **Feature Engineering** projects on real-world datasets from food, retail, and travel domains.

---

## 📁 Project Structure

EDA/
├── problem_1(Zomato)/
├── problem_2(Black_Friday)/
└── problem_3(Flight_Price)/

---

## 🍽️ Zomato EDA

- Handled missing values in ratings, cost, and cuisines
- Analyzed top cuisines and restaurant types
- Explored city-wise trends
- Visualized rating distributions and cost vs votes

---

## 🛍️ Black Friday Sales EDA

- Handled missing values in product category features
- Applied encoding on categorical variables (Gender, Age, City Category)
- Analyzed customer purchase behavior across demographics

---

## ✈️ Flight Price EDA & Feature Engineering

- Extracted date features (Day, Month, Year) from `Date_of_Journey`
- Converted time features (`Dep_Time`, `Arrival_Time`) into numerical columns
- Transformed `Duration` into total minutes
- Encoded categorical features (Airline, Source, Destination, Additional_Info)
- Converted `Total_Stops` into numerical format
